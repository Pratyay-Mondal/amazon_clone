### **Title:** 
Full-Stack E-Commerce Platform (Amazon Clone)

### **Description:**
*   **Problem:** Addressed the complexities of managing dynamic e-commerce state, such as real-time cart synchronization and secure user authentication across decoupled client-server boundaries.
*   **Solution:** Engineered a scalable RESTful architecture using FastAPI for rapid backend data routing and JWT security, coupled with a React and Zustand frontend styled with Tailwind CSS to replicate industry-standard UX behavior.
*   **Result:** Delivered a fully functional, highly responsive web application featuring a populated product catalog, detailed item inspection, and an end-to-end simulated checkout workflow.

### **Tech Section:**
*   **Frontend:** React, TypeScript, Vite, Tailwind CSS v4, Zustand (State Management), React Router
*   **Backend:** Python, FastAPI, SQLAlchemy (ORM), Pydantic, Passlib (bcrypt), python-jose (JWT)
*   **Database/Architecture:** SQLite, RESTful API Design, Component-Driven Architecture
